# Architecture Technique

## 1. Frontend
-   **Framework :** React avec TypeScript pour un code robuste et typé.
-   **Cible :** Application web responsive avec une approche **Mobile First**.
-   **Technologie :** Doit être conçue comme une **Progressive Web App (PWA)**. Cela permet une installation sur l'écran d'accueil des mobiles et un accès hors-ligne limité, offrant une expérience proche d'une application native sans passer par les app stores.

## 2. Backend
-   **Framework :** Node.js avec un framework comme Express.js pour la création de l'API REST.
-   **Langage :** TypeScript pour la cohérence avec le frontend.

## 3. Base de Données
-   **Type :** Base de données NoSQL pour sa flexibilité, comme **MongoDB** ou **Firestore (Google Firebase)**. Ce choix est idéal pour stocker des documents structurés comme les profils utilisateurs et la base de connaissance.
