# CaniCoach-IA

## Mission du Projet
CaniCoach IA est une application mobile conçue pour rendre l'éducation canine éthique, personnalisée et accessible à tous, 24/7. Notre mission est de renforcer la relation entre les humains et leurs chiens pour prévenir les problèmes de comportement et les abandons.

## Philosophie
Le projet est entièrement basé sur les méthodes et la philosophie de l'éducateur canin **Tony Silvestre (Esprit Dog)**. L'approche est centrée sur la bienveillance, la patience, la déculpabilisation du propriétaire et la compréhension scientifique du comportement canin.

## Description du Produit
L'application est un assistant conversationnel (chatbot IA) qui fournit des conseils personnalisés aux propriétaires de chiens. Elle s'adresse aussi bien aux nouveaux propriétaires de chiots qu'aux maîtres de chiens adultes ayant des lacunes éducatives.

Le modèle économique est basé sur un abonnement mensuel/annuel avec une période d'essai gratuite permettant de résoudre un premier problème.

## Structure du Projet
Ce repository sert de "source de vérité" pour le développement de l'application.

-   Le dossier `/Cahier_des_Charges` contient les spécifications fonctionnelles et techniques détaillées.
-   Le dossier `/IA_Instructions` contient les directives pour la personnalité de l'IA et des exemples pour sa base de connaissance.
